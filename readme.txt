Add downloadable attachments to WooCommerce products.

== Description ==

Product Attachment for WooCommerce allows you to add downloadable attachments to your WooCommerce products. You can add multiple attachments and display them in a product tab or using a shortcode.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/product-attachment-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Settings > Product Attachments to configure the plugin.

== Frequently Asked Questions ==

= How do I add attachments to a product? =

Go to the product edit screen and you will see a new meta box called "Product Attachments". You can add your attachments there.

= How do I display attachments on the frontend? =

You can display attachments in a product tab or using the `[product_attachments]` shortcode.
